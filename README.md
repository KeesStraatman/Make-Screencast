# Make Screeencast


I took the original code included with Fiji and the error messages displayed and asked Google Gemini to resolve the problem. This was repeated several times till a working version was created.

Save the bsh file under Fiji > Scripts > File and it will replace the old version. If you want to keep the old version, just rename it. Once installed, you will find the option in Fiji under File/ Make Screencast.

Requires the free VLC media player (https://www.videolan.org/)
